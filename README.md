# Place Blend Recipe Builder

A no-API Streamlit app that creates:

- a structured JSON image recipe
- a 3:2 composition guide PNG
- a locked ChatGPT image-generation prompt

## Deploy on Streamlit Community Cloud

1. Upload `app.py` and `requirements.txt` to the repository root.
2. In Streamlit Community Cloud choose:
   - Repository: this repository
   - Branch: `main`
   - Main file path: `app.py`
3. Deploy.

No OpenAI API key is required.
