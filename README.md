# Place Blend Maker — Clean Local Travel Art

Build 2.1 converts ordinary place photos into coordinated illustrated travel-poster images, then combines them into posters, postcards and collages.

The app runs locally on Streamlit using Pillow, NumPy and OpenCV. It does not use a paid image-generation API.

## Deploy
Upload every file and the `.streamlit` folder to GitHub, then deploy `app.py` on Streamlit Community Cloud.

## Recommended first test
- Engine: Clean reference-guided illustration
- Preset: Painterly Travel Poster
- Strength: 55–65
- Quality: Balanced
- Match all photos to reference colours: on
