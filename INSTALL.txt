TRAVEL ART RECIPE BUILDER — INSTALL

ADD TO YOUR EXISTING STREAMLIT APP
1. Put 3_Travel_Art_Recipe_Builder.py inside your app's pages folder.
2. Add these to requirements.txt if they are not already included:
   streamlit>=1.32
   Pillow>=10.0
3. Commit to GitHub and redeploy.

STANDALONE USE
Run:
streamlit run 3_Travel_Art_Recipe_Builder.py

THIS VERSION
- Uses no OpenAI API.
- Creates no separate image-generation charge.
- Generates a JSON recipe, visual 3:2 composition guide and final prompt.
- Includes presets for Melbourne, Amalfi Coast, Santorini, Sicily, Sydney, Paris, Rome and Venice.
- Unknown destinations receive an editable generic template.
